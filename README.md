Personal portfolio hub for Nitin Tyagi — hosted at [tyaginboston.github.io](https://tyaginboston.github.io).

## What's here

- **Live project thumbnails** (iframe previews) for open-source work
- **Professional experience** cards for MassMutual, Cigna, and myCigna
- Zero build step — pure HTML + CSS, deploys directly via GitHub Pages

## Deployment

Push to `main` → GitHub Pages serves `index.html` automatically from the root of a `<username>.github.io` repository. No Actions workflow needed.
