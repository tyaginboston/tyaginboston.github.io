## About

This portfolio showcases 12+ years of frontend engineering experience across Fortune 500 companies in insurance and healthcare, including MassMutual, Cigna, and The Hanover Insurance Group.

## Built With

- Vanilla HTML, CSS, JavaScript — no framework, no build step
- Hosted on GitHub Pages
- Single `index.html` — fully self-contained

## Sections

- **Hero** — Introduction and quick links
- **Work** — Open source projects and enterprise work
- **Skills** — Full tech stack
- **Experience** — Career timeline
- **Contact** — Get in touch

## Local Development

No build step needed. Just open `index.html` in a browser:

```bash
git clone https://github.com/tyaginboston/tyaginboston.github.io
open index.html
```
